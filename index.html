What is the Difference between SGPA and CGPA
SGPA is for a particular semester and CGPA is for accumulated all semesters
How do we balance CGPA, DSA and Web Development in first year
You should focus on clearing the basics of DSA in first year ad then start solving leetcode. Development can done in 2nd year as well and you can do it also on the weekend by follwing DSA on weekdays. For CGPA, you should start preparing before 2 weeks of exam.
How should I start my web development journey
Start with basic Html and CSS form youtube and then make small level projects after that shift to JS
What is best way to study for college exams
Attend classes to know the important topics and then cover them up before exams from youtube
